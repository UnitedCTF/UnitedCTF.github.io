# unitedctf.github.io

> https://unitedctf.github.io

UnitedCTF website.


## requirements

- ruby
- gem bundler

## developement

Install dependencies:

```bash
bundle install
```

Start server:

```bash
bundle exec jekyll serve
```

Open your browser at [http://127.0.0.1:4000](http://127.0.0.1:4000).
